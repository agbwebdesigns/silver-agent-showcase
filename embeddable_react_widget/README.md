# Silver Agent Showcase

Silver Agent is an embeddable AI assistant platform designed for Medicare agencies. It provides general Medicare education through a website chat widget while following compliance-aware rules around privacy, plan recommendations, and lead handling.

This repository is a public showcase version. The production codebase is private. Proprietary prompt logic, customer validation, deployment configuration, credentials, and compliance-sensitive implementation details have been removed.

## Project Goals

- Provide general Medicare education through an agency website
- Offer an embeddable AI chat widget
- Support agency-specific metadata
- Build toward compliant lead qualification workflows
- Prepare for future broker portal and audit logging
